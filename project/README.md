# Saint Joseph School Website

A simple, modern website for Saint Joseph School built with pure HTML, CSS, and JavaScript.

## Features

- Modern, responsive design
- Smooth scrolling navigation
- Dynamic header effects
- Interactive enrollment system
- Animated counters
- Mobile-friendly layout

## How to Run

1. Simply open the `index.html` file in your web browser
   - Double click the file
   - Or drag and drop it into your browser
   - Or use a local server (e.g., Python's `python -m http.server` or VS Code's Live Server)

## Project Structure

- `index.html` - Main webpage
- `style.css` - All styling and animations
- `main.js` - Core JavaScript functionality
- `counter.js` - Counter animations

## Browser Compatibility

Tested and working on:
- Chrome
- Firefox
- Safari
- Edge
